# Nomos UI - 独立フロントエンドリポジトリ

Nomos DAO Frameworkの完全に独立したフロントエンド専用リポジトリ。React + TypeScript + Tailwind CSSで実装されたモジュラー型DAOの可視化・管理インターフェース。

## リポジトリ目的とアーキテクチャ概要

### 目的
nomos-uiは、Nomos DAO Frameworkエコシステムにおける完全に独立したフロントエンド専用リポジトリです：

- **独立性**: バックエンド（nomos-core）から完全に分離された開発・デプロイメント体制
- **モジュラー設計**: 再利用可能なコンポーネントベースのアーキテクチャ
- **拡張性**: 新機能やエンジンタイプへの柔軟な対応
- **貢献しやすさ**: フロントエンド開発者が独立して貢献できる環境
